6- Convergency settings

This file gives the input variables that govern the convergency of ground state calculations in ABINIT.

The convergency of the energy must be tested towards the plane wave convergency and the k-point convergency according to tolerance criteria. The convergency of the Self-Consistent cycle is governed by the way densities or potentials are mixed between each iteration.
The "accuracy" variable enables to tune the accuracy of a calculation by setting automatically up to seventeen variables.

    3. Related input variables.

    Compulsory input variables:

    ... ecut [Energy CUToff]
    ... pawecutdg [PAW - Energy CUToff for the Double Grid]

    Basic input variables:

    ... accuracy [ACCURACY]
    ... ecutsm [Energy CUToff SMearing]
    ... ngfft [Number of Grid points for Fast Fourier Transform]
    ... nstep [Number of (non-)self-consistent field STEPS]
    ... toldfe [TOLerance on the DiFference of total Energy]
    ... tolrff [TOLerance on the Relative diFference of Forces]
    ... tolvrs [TOLerance on the potential V(r) ReSidual]
    ... tolwfr [TOLerance on WaveFunction squared Residual]

    Useful input variables:

    ... diegap [DIElectric matrix GAP]
    ... dielam [DIElectric matrix LAMbda]
    ... dielng [model DIElectric screening LeNGth]

    ... diemix [model DIElectric MIXing factor]
    ... iprcel [Integer for PReConditioning of ELectron response]
    ... iscf [Integer for Self-Consistent-Field cycles]
