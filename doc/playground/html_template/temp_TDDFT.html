3- TDDFT

This file explains how to perform TDDFT calculation with the ABINIT package.

For finite systems (atoms and molecules), excited states can be computed within TDDFT (Casida approach - only norm-conserving pseudopotentials). See the explanations given in the tutorial. The iscf input variable must be set to -1.
