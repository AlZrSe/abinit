1. Related input variables.

Input variables related to alchemical mixing:

... algalch [ALGorithm for generating ALCHemical pseudopotentials]
... mixalch [MIXing coefficients for ALCHemical potentials]
... npsp [Number of PSeudoPotentials]
... %npspalch [Number of PSeudoPotentials that are "ALCHemical"]
... ntypalch [Number of TYPe of atoms that are "ALCHemical"]
... %ntyppure [Number of TYPe of atoms that are "PURe"]

Input variables for job time limits:

... cpuh [CPU time limit in Hours]
... cpum [CPU time limit in Minutes]
... cpus [CPU time limit in seconds]

Input variables to insert a slab:

... jellslab [include a JELLium SLAB in the cell]
... slabwsrad [jellium SLAB Wigner-Seitz RADius]
... slabzbeg [jellium SLAB BEGinning edge along the Z direction]
... slabzend [jellium SLAB ENDing edge along the Z direction]

Input variables for Learn On The Fly calculations:

... lotf_classic [LOTF CLASSIC model for glue model]
... lotf_nitex [LOTF Number of ITerations]
... lotf_nneigx [LOTF max Number of NEIGhbours]
... lotf_version [LOTF VERSION of MD algorithm]

Input variables for experts:

... builtintest [BUIT-IN TEST number]
... chempot [spatially varying CHEMical POTential]
... chkexit [CHecK whether the user want to EXIT]
... einterp [Electron bands INTERPolation]
... eshift [Energy SHIFT]
... fftcache [Fast Fourier Transform CACHE size]
... istatr [Integer for STATus file rate]
... istatshft [Integer for STATus file SHiFT]
... mem_test [MEMory TEST]
... nloc_alg [Non LOCal ALGorithm]
... nloc_mem [Non LOCal MEMOry]
... nzchempot [Number of Z reduced coordinates that define the spatial CHEMical POTential]
... papiopt [PAPI OPTion]
... random_atpos [RANDOM ATomic POSitions]
... usepotzero [USE POTential ZERO]
... useria [USER Integer variable A]
... userib [USER Integer variable B]
... useric [USER Integer variable C]
... userid [USER Integer variable D]
... userie [USER Integer variable E]
... userra [USER Real variable A]
... userrb [USER Real variable B]
... userrc [USER Real variable C]
... userrd [USER Real variable D]
... userre [USER Real variable E]
