1. Related lesson(s) of the tutorial.

  • The additional information given by lesson RF1 opens the door to The lesson on Optic, the utility that allows to obtain the frequency dependent linear optical dielectric function and the frequency dependent second order nonlinear optical susceptibility, in the simple "Sum-Over-State" approximation.
    •  
     

    2. Related input variables.

    Basic input variables:

    ... broadening@optic [BROADENING]
    ... ddkfile@optic [DDK FILE]
    ... domega@optic [Delta OMEGA]
    ... lin_comp@optic [LINear COMPonents]
    ... maxomega@optic [MAXimum valur of OMEGA]
    ... nonlin_comp@optic [NON-LINear COMPonents]
    ... num_lin_comp@optic [NUMber of LINear COMPonents]
    ... num_nonlin_comp@optic [NUMber of NON-LINear COMPonents]
    ... scissor@optic [SCISSOR operator]
    ... tolerance@optic [TOLERANCE]
    ... wfkfile@optic [WaveFunction K FILE]
