1. Introduction.

This ANADDB utility is able to perform many different tasks, each governed by a selected set of input variables, with also some input variables common to many of the different tasks. The list of tasks to be done in one run is governed by different flags. Here is the list of flags :

  • dieflag Calculation of the dielectric tensor. 
        related input variables: nfreq frmax frmin
  • elaflag Calculation of the elastic tensor
  • elphflag Electron-phonon calculations
  • ifcflag Calculation of the interatomic force constants 
        related input variables: atifc dipdip ifcana ifcout natifc nsphere prt_ifc rifcsph vs_qrad_tolkms
  • instrflag Calculation of the internal strain tensor
  • nlflag Calculation of the non-linear susceptibilities 
        related input variables: alphon  prtmbm ramansr
  • piezoflag Calculation of piezoelectric tensors
  • polflag Compute constrained polarization 
        related input variables: natfix  iatfix nstrfix istrfix relaxat relaxstr targetpol
  • thmflag Calculation of thermal quantities 
        related input variables: gruns_ddbs gruns_nddbs ntemper temperinc tempermin thmtol

 

 

2. Related lesson(s) of the tutorial.

    3. Related input variables.

    Compulsory input variables:

    ... dieflag@anaddb [DIElectric FLAG]
    ... elaflag@anaddb [ELAstic tensor FLAG]
    ... ifcflag@anaddb [Interatomic Force Constants FLAG]
    ... instrflag@anaddb [INternal STRain FLAG]
    ... nlflag@anaddb [Non-Linear FLAG]
    ... piezoflag@anaddb [PIEZOelectric tensor FLAG]
    ... polflag@anaddb [POLarization FLAG]
    ... thmflag@anaddb [THerMal FLAG]

    Basic input variables:

    ... alphon@anaddb [ALign PHONon mode eigendisplacements]
    ... atifc@anaddb [AToms for IFC analysis]
    ... frmax@anaddb [FRequency MAXimum]
    ... frmin@anaddb [FRequency MINimum]
    ... gruns_ddbs@anaddb [GRUNeiSen DDBS]
    ... gruns_nddbs@anaddb [GRUNeiSen Number of DDB files]
    ... iatfix@anaddb [Indices of the AToms that are FIXed]
    ... ifcana@anaddb [IFC ANAlysis]
    ... ifcout@anaddb [IFC OUTput]
    ... istrfix@anaddb [Index of STRain FIXed]
    ... natfix@anaddb [Number of AToms FIXed]
    ... natifc@anaddb [Number of AToms for IFC analysis]
    ... nfreq@anaddb [Number of FREQuencies]
    ... nstrfix@anaddb [Number of STRain components FIXed]
    ... ntemper@anaddb [Number of TEMPERatures]
    ... prt_ifc@anaddb [PRinT the Interatomic Force Constants]
    ... prtmbm@anaddb [PRinT Mode-By-Mode decomposition of the electrooptic tensor]
    ... ramansr@anaddb [RAMAN Sum-Rule]
    ... relaxat@anaddb [RELAXation of AToms]
    ... relaxstr@anaddb [RELAXation of STRain]
    ... rifcsph@anaddb [Radius of the Interatomic Force Constant SPHere]
    ... targetpol@anaddb [TARGET POLarization]
    ... temperinc@anaddb [TEMPERature INCrease]
    ... tempermin@anaddb [TEMPERature MINimum]
    ... thmtol@anaddb [THerModynamic TOLerance]

    Useful input variables:

    ... dipdip@anaddb [DIPole-DIPole interaction]
    ... nsphere@anaddb [Number of atoms in SPHERe]
    ... vs_qrad_tolkms@anaddb [Speed of Sound Q-radius, TOLerance KiloMeter/Second]
