1. Introduction.

All the files that have the density and potential format, see topic_printing can be analyzed with the "Cut3D" postprocessor. In particular, it can produce two-dimensional cuts through the three-dimensional data, suitable for later visualisation using e.g. Abipy.

See the [[FAKE LINK:FAKE LINK:help_cut3D]], as well as the [[FAKE LINK:FAKE LINK:tutorial_AnalysisTools]].
