1. Introduction.

ABINIT is developed under the control of git, a well-known Version Control System. There is an official github mirror of the Abinit repository. We welcome bug fixes and improvements. Note that most of the active developments are hosted on the Gitlab server. Before embarking on making significant changes, please contact the Abinit group.

See the many different projects linked to ABINIT on Github, actually gathered in the Github ABINIT organization.
