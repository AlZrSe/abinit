Convergency settings

This file gives hints on how to set parameters to tune the convergency of a calculation with the ABINIT package.

1. Introduction.

The convergency of the energy must be tested towards the plane wave convergency and the k-point convergency according to tolerance criteria. The convergency of the Self-Consistent cycle is governed by the way densities or potentials are mixed between each iteration.
The accuracy variable enables to tune the accuracy of a calculation by setting automatically up to seventeen variables.
At the level of the generation of electronic eigenfunctions, an efficient band-by-band preconditioned conjugate-gradient algorithm is used, in its non-self-consistent version. An alternative algorithm "lobpcg" is also available.
At the level of the self-consistency loop, an efficient potential-based preconditioned conjugate-gradient algorithm is used. Simple mixing is also available, as well as the Anderson algorithm. Preconditioning of this algorithm is achieved through a model dielectric function, or through an approximate dielectric matrix.

    3. Related input variables.

    Compulsory input variables:

    ... ecut [Energy CUToff]
    ... pawecutdg [PAW - Energy CUToff for the Double Grid]

    Basic input variables:

    ... accuracy [ACCURACY]
    ... ecutsm [Energy CUToff SMearing]
    ... ngfft [Number of Grid points for Fast Fourier Transform]
    ... nstep [Number of (non-)self-consistent field STEPS]
    ... toldfe [TOLerance on the DiFference of total Energy]
    ... tolrff [TOLerance on the Relative diFference of Forces]
    ... tolvrs [TOLerance on the potential V(r) ReSidual]
    ... tolwfr [TOLerance on WaveFunction squared Residual]

    Useful input variables:

    ... diecut [DIElectric matrix energy CUToff]
    ... diegap [DIElectric matrix GAP]
    ... dielam [DIElectric matrix LAMbda]
    ... dielng [model DIElectric screening LeNGth]
    ... diemac [model DIElectric MACroscopic constant]
    ... diemix [model DIElectric MIXing factor]
    ... iprcel [Integer for PReConditioning of ELectron response]
    ... iscf [Integer for Self-Consistent-Field cycles]
    ... npulayit [Number of PULAY ITerations for SC mixing]

    Relevant internal variables:

    ... %mgfft [Maximum of nGFFT]
    ... %mgfftdg [Maximum of nGFFT for the Double Grid]
    ... %mpw [Maximum number of Plane Waves]
    ... %nfft [Number of FFT points]
    ... %nfftdg [Number of FFT points for the Double Grid]

    Input variables for experts:

    ... boxcutmin [BOX CUT-off MINimum]
    ... densfor_pred [DENSity and FORces PREDictor]
    ... exchn2n3d [EXCHange N2 and N3 Dimensions]
    ... extrapwf [flag - EXTRAPolation of the Wave-Functions]
    ... fftalg [Fast Fourier Transform ALGorithm]
    ... iboxcut [Integer governing the internal use of BOXCUT - not a very good choice of variable name]
    ... icoulomb [Index for the Coulomb TReaTMenT]
    ... iprcfc [Integer for PReConditioner of Force Constants]
    ... isecur [Integer for level of SECURity choice]
    ... mqgrid [Maximum number of Q-space GRID points for pseudopotentials]
    ... nbdblock [Number of BanDs in a BLOCK]
    ... nbdbuf [Number of BanDs for the BUFfer]
    ... nc_xccc_gspace [Norm-Conserving pseudopotentials - use XC Core-Correction in G-SPACE]
    ... nline [Number of LINE minimisations]
    ... nnsclo [Number of Non-Self Consistent LOops]
    ... nscforder [Nth - SCaling Function ORDER]
    ... ortalg [ORThogonalisation ALGorithm]
    ... tolrde [TOLerance on the Relative Difference of Eigenenergies]
