1. Introduction

The convergency of the energy must be tested towards the plane wave convergency and the k-point convergency according to tolerance criteria. The convergency of the Self-Consistent cycle is governed by the way densities or potentials are mixed between each iteration.
The "accuracy" variable enables to tune the accuracy of a calculation by setting automatically up to seventeen variables.

 

 

2.Input variable settings.

Global convergency input variable: accuracy

Plane wave convergency input variables: ecut ecutsm ngfft

Self-Consistent-Field convergency and mixing input variables: diecut diegap dielam dielng diemac diemix iprcel iscf npulayit

Tolerance criterium for convergency input variables: nstep toldfe toldff tolrff tolvrs tolwfr

Input variables for experts:

boxcutmin densfor_pred exchn2n3d extrapwf fftalg iboxcut icoulomb iprcfc isecur mqgrid nc_xccc_gspace nbdblock nbdbuf nline nscforder nnsclo ortalg tolrde tolsym wfoptalg
