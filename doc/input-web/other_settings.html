1. Introduction

In this section, some very specific input variables are gathered, either to perform alchemical calculations, to build slabs, to control job time limits or for expert developpers.

 

 

2.Input variable settings.

Input variables related to alchemical mixing:

algalch mixalch npsp npspalch ntypalch ntyppure

Input variables for job time limits:

cpuh cpum cpus

Input variables to insert a slab:

jellslab slabwsrad slabzbeg slabzend

Input variables for experts:

eshift nloc_alg nloc_mem papiopt random_atpos usepotzero

Input variables for expert developpers:

mem_test builtintest fftcache istatr istatshft useria userib useric userid userie userra userrb userrc userrd userre
